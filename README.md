# Arume-pengering_sepatu
ARUME (Automatic Shoe Dryer) adalah alat pengering sepatu berbasis Internet of Things (IoT) yang menggunakan ESP32, Soil Moisture Sensor, dan DHT22 untuk memantau kelembapan serta suhu secara real-time. Sistem ini terhubung dengan aplikasi Android berbasis Flutter melalui protokol MQTT (HiveMQ Cloud),
(flutter pub get ) Jika ingin mengembalikan file yang sudah kami clear.
